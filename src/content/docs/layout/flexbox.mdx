title: Flexbox
---

Component for building flexible box layouts.

<Example gap="8">
  <Column alignItems="center">
    <Label>Row</Label>
    <Flexbox direction="row" gap="3">
      <ExampleBox borderSize="1" />
      <ExampleBox borderSize="1" />
      <ExampleBox borderSize="1" />
    </Flexbox>
  </Column>
  <Column>
    <Label>Column</Label>
    <Flexbox direction="column" gap="3">
      <ExampleBox borderSize="1" />
      <ExampleBox borderSize="1" />
      <ExampleBox borderSize="1" />
    </Flexbox>
  </Column>
</Example>
