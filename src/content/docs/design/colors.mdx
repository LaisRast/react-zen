Our color system is based off of [Radix Colors](https://www.radix-ui.com/colors). There are several categories of colors that we use in this library and they are all defined in CSS variables.

## Base colors

Base colors are used as the basis of every UI element including backgrounds, borders, and text.

## CSS variables

The base colors are a 12-step color range going from lightest to darkest.
