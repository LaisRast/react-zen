There are several categories of colors that we use in this library and they are all defined in CSS variables.

## Base colors

Base colors are used as the basis of every UI element including backgrounds, borders, and text.

## CSS variables

The base colors are a 11-step color range going from lightest to darkest.
We used the colors created by [Tailwind](https://tailwindcss.com/docs/customizing-colors), so you can
easily swap these out with any of their other color palettes.
