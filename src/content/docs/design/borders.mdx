## Border radius

Border radius values are defined as a 6-step scale from smallest to largest.

  <Label>Radius 1</Label>
  <ExampleBox borderSize="1" borderRadius="1" />
  <Label>Radius 2</Label>
  <ExampleBox borderSize="1" borderRadius="2" />
  <Label>Radius 3</Label>
  <ExampleBox borderSize="1" borderRadius="3" />
  <Label>Radius 4</Label>
  <ExampleBox borderSize="1" borderRadius="4" />
  <Label>Radius 5</Label>
  <ExampleBox borderSize="1" borderRadius="5" />
  <Label>Radius 6</Label>
  <ExampleBox borderSize="1" borderRadius="6" />
</Example>

## Border size

Border sizes are defined as a 4-step scale from smallest to largest.

<Example gap="5">
  <Label>Size 1</Label>
  <ExampleBox borderSize="1" borderRadius="3" />
  <Label>Size 2</Label>
  <ExampleBox borderSize="2" borderRadius="3" />
  <Label>Size 3</Label>
  <ExampleBox borderSize="3" borderRadius="3" />
  <Label>Size 4</Label>
  <ExampleBox borderSize="4" borderRadius="3" />
</Example>
