# Introduction

**zen** is a modern, minimalist design system created by [Umami](https://umami.is).

**react-zen** is a React component library implementing the zen design system. It is built on top
of [React Aria](https://react-spectrum.adobe.com/react-aria/) and [Radix](https://www.radix-ui.com/primitives) primitives
and simply styled with pure CSS and CSS variables.
