# Text

A text element.

<Example>
  <Text>Here is some text.</Text>
</Example>

## Colors

<Example>
  <Text>Normal text</Text>
  <Text color="muted">Muted text</Text>
  <Text color="faded">Faded text</Text>
</Example>

## Sizes

  <Text size={2}>Text 2</Text>
  <Text size={3}>Text 3</Text>
  <Text size={4}>Text 4</Text>
  <Text size={5}>Text 5</Text>
  <Text size={6}>Text 6</Text>
</Example>

## Weights

<Example>
  <Text weight="light">Light text</Text>
  <Text>Normal text</Text>
  <Text weight="bold">Bold text</Text>
  <Text weight="bolder">Bolder text</Text>
</Example>
