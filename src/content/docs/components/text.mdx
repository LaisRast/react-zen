# Text

A text element.

<Example>
  <Text>Here is some text.</Text>
</Example>

## Types

<Example>
  <Text>Normal text</Text>
  <Text type="muted">Muted text</Text>
  <Text type="faded">Faded text</Text>
</Example>

## Sizes

    <Text size="md">Medium text</Text>
    <Text size="lg">Large text</Text>

  </Column>
</Example>

## Weights

<Example>

  <Text weight="light">Light text</Text>

</Example>
