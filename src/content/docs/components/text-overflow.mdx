# Text Overflow

Truncates long text with ellipsis.

<Example>
  <TextOverflow>This is some really long text that should truncate.</TextOverflow>
  <TextOverflow style={{maxWidth: 200}}>This is some really long text that should truncate.</TextOverflow>
  <TextOverflow style={{maxWidth: 100}}>This is some really long text that should truncate.</TextOverflow>
</Example>
