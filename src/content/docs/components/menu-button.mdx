# Menu Button

A button that displays a dropdown menu.

<Example>
  <MenuButton>
    <Menu items={['One', 'Two', 'Three', 'Four', 'Five']} />
  </MenuButton>
</Example>

## Custom button

<Example>
  <MenuButton button={<Button variant="primary">Button</Button>}>
    <Menu items={['One', 'Two', 'Three', 'Four', 'Five']} />
  </MenuButton>
</Example>
