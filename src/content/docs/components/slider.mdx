# Slider

A slider allows a user to select one or more values within a range.

<Example>
  <Slider label="Opacity" defaultValue={50} style={{ maxWidth: 300 }} />
</Example>

## Orientation

<Example>
  <Slider orientation="vertical" label="Opacity" defaultValue={50} style={{ maxWidth: 300 }} />
</Example>
