# Toggle Group

A set of two-state buttons that can be toggled on or off.

<Example>
  <ToggleGroup type="single">
    <ToggleGroupItem value="one">One</ToggleGroupItem>
    <ToggleGroupItem value="two">Two</ToggleGroupItem>
    <ToggleGroupItem value="three">Three</ToggleGroupItem>
  </ToggleGroup>
</Example>

## Multiple select

<Example>
  <ToggleGroup type="multiple">
    <ToggleGroupItem value="one">One</ToggleGroupItem>
    <ToggleGroupItem value="two">Two</ToggleGroupItem>
    <ToggleGroupItem value="three">Three</ToggleGroupItem>
  </ToggleGroup>
</Example>

## Icons

<Example>
  <ToggleGroup type="single">
    <ToggleGroupItem value="one"><AlignLeft /></ToggleGroupItem>
    <ToggleGroupItem value="two"><AlignCenter /></ToggleGroupItem>
    <ToggleGroupItem value="three"><AlignRight /></ToggleGroupItem>
  </ToggleGroup>
</Example>
