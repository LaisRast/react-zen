# Text Field

A text field allows a user to enter a plain text value with a keyboard.

<Example>
  <TextField />
</Example>

## Placeholder text

<Example>
  <TextField placeholder="Enter some text" />
</Example>

## As textarea

<Example>
  <TextArea placeholder="Enter some text" />
</Example>

## Disabled

<Example>
  <TextField defaultValue="Disabled" isDisabled />
</Example>

## Read only

<Example>
  <TextField defaultValue="Read only" isReadOnly />
</Example>

## Copy button

<Example>
  <TextField value="Here is some text." allowCopy={true} />
  <TextArea value="Here is some text." allowCopy={true} />
</Example>

## With label

<Example>
  <TextField label="Name" placeholder="Enter your name." />
</Example>
